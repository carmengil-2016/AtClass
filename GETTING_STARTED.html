Welcome to Your Math Learning App!

  1. Open index.html in your browser
  2. Explore the different sections (Home, Topics, Progress)

Next Steps

1. Start the App

Open index.html directly in your browser, or run a local server:

# Python 3
python -m http.server 8000

# Then open: http://localhost:8000

2. Customize Your App

  • Change Colors: Edit CSS variables in styles.css
  • Update Topics: Modify the topics grid in index.html
  • Add Lessons: Update assets/data/lessons.json
  • Change App Name: Update in index.html and manifest.json

3. Add Content

The assets/data/lessons.json file contains sample lesson data:

  • Topics: Arithmetic, Fractions, Algebra, Geometry
  • Lessons: Structured learning content
  • Exercises: Quiz questions with multiple choice answers

4. Test Offline Functionality

  1. Open DevTools (F12 / Cmd+Option+I)
  2. Go to "Application" tab
  3. Check "Offline" under Service Workers
  4. Reload the page - it should still work!

5. Install as App

Mobile (Android): Open in Chrome → Menu (⋮) → "Add to Home Screen"

Mobile (iOS): Open in Safari → Share → "Add to Home Screen"

Desktop: Click the install icon in your browser's address bar

Key Features Explained

Progress Tracking

Access progress from the browser console:

// View progress
AtClass.progressTracker.data

// Complete a lesson
AtClass.progressTracker.completeLesson()

// Reset progress
AtClass.resetProgress()

Mobile Navigation

  • Hamburger menu on mobile devices
  • Full navigation on larger screens
  • Smooth section transitions
  • Active link highlighting

Responsive Design

  • Mobile: Stack layout, single column
  • Tablet: 2-column grid for topics
  • Desktop: 3-column grid with expanded layout

Customization Tips

Change Primary Color

In styles.css, find the :root section and update:

--primary-color: #YOUR_COLOR_HERE;
--secondary-color: #YOUR_ACCENT_COLOR;

Add New Topics

In index.html, find the topics grid and duplicate a topic-card div with your new content.

Update Lesson Data

Edit assets/data/lessons.json to add new lessons and exercises.

Browser Support

  • ✅ Chrome / Edge: Full support
  • ✅ Firefox: Full support
  • ✅ Safari: Full support
  • Older browsers: Core functionality works, some advanced features may be limited
